Important Features:
Save and Load the game from any point. Can also have multiple saves. Last used directory is remembered and automatically selected.
Game will warn player when trying to do something invalid, like entering early, entering a fake word, or deleting when empty.
Can clear the board/get a new word at any point, while keeping stats.
Can return the main menu, and continue from their while keeping stats.
Follows object oriented programming, uses collections and inheritence.
Uses a HashMap and a custom data structure to store and save the board.
Keyboard is updated to help player know what letters to use.
Player cannot change previous answers, and all wordle game rules are followed (with warnings to the player).
File IO for words and saves.
Input validation for the saves. Game will not crash on a bad read, just warn the player and let them try again.
